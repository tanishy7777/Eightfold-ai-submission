🎬 Presenter Mode — invisible teleprompter

A separate floating window holds your demo script and auto-scrolls. The trick to keeping it out of the recording: record only the diagram, not the whole screen.

1
Open the teleprompter — a small always-on-top window appears (Picture-in-Picture). Drag it anywhere off to the side.
2
Record a region, not the screen. macOS: ⌘⇧5 → "Record Selected Portion" → draw the box around only this diagram. The teleprompter sits outside the box → never captured.
3
Or capture a single window in OBS/QuickTime (Window Capture → this browser window). The PiP teleprompter is a different window, so it stays invisible.
⚠️ Honest note: no webpage can hide a div from a full-screen recorder (that's OS-level DRM, like Netflix). This works by putting the script in a separate window you don't record — the reliable way pros run hidden teleprompters.
